# /sharing/add_file_member

### Description
Adds specified members to a file.

### URL Structure
```
https://api.dropbox.com/1/sharing/add_file_member
```

import LinkedTypeExplanation from '../../components/LinkedTypeExplanation'

### parameters
Example
```
{
    "access_level": "viewer",
    "add_message_as_comment": false,
    "custom_message": "This is a custom message about ACME.doc",
    "file": "id:3kmLmQFnf1AAAAAAAAAAAw",
    "members": [
        {
            ".tag": "email",
            "email": "justin@example.com"
        }
    ],
    "quiet": false
}
```
