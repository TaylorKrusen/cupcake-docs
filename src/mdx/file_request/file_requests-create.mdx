# /file_requests/create

### Description
Creates a file request for this user.

### URL Structure
```
https://api.dropbox.com/1/file_requests/create
```

import LinkedTypeExplanation from '../../components/LinkedTypeExplanation'

### parameters
Example
```
{
    "deadline": {
        "allow_late_uploads": "seven_days",
        "deadline": "2020-10-12T17:00:00Z"
    },
    "destination": "/File Requests/Homework",
    "open": true,
    "title": "Homework submission"
}
```
